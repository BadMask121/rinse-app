## Rinse Orders Map 

First, run the development server:

```bash
npm run dev
# or
yarn dev
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.

## Api can be access at 
can be accessed on [http://localhost:3000/api/orders](http://localhost:3000/api/orders). 
This endpoint can be edited in `pages/api/orders/[date].ts`.


## Api Documentation endpoint 
can be accessed on [http://localhost:3000/doc](http://localhost:3000/doc).
